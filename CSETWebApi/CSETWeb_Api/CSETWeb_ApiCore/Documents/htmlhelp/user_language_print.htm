User Language

 
Selecting the User Language option from the User Profile dropdown menu opens the User Language window.
 

User Language

  • Use the dropdown field to select one of the following available user languages. Translations are only available for the specified assessments.
  • English (Default)
  • Spanish
  • Automated Cybersecurity Evaluation Toolbox (ACET)
  • Ukrainian
  • Cybersecurity Framework 2.0 (CSF 2.0)
  • Translation is available only in Requirements Mode
  • Cybersecurity Framework 1.1 (CSF 1.1)
  • Translation is available only in Requirements Mode
  • Ransomware Readiness Assessment (RRA)
  • Cybersecurity Performance Goals (CPG)

OK

  • Select the OK button to save the User Language selection.

Cancel

  • Select the Cancel button to close the User Language window without enabling any changes.
