Network Component Overrides

 
 
Override default answers for component types or individual components in the Question Information panel. Click "Override" next to a specific component type to enter a different answer. If the option is "None," it means the question relates to that component type, but there are no instances on the diagram. Clicking "Override" opens the Network Component Overrides screen, where you can answer the question for all instances of that component type. Later, you can customize individual answers for specific devices. This example pertains to overriding the default answer for an IDS.
 
Figure: Network Component Overrides screen
 
Here, you answer the question for all instances of that component type. Later, you can change any single answer to match a particular device. In this example, the screen relates to overriding the default answer for an IDS.
 
Remember:
1. Any specifically answered question using the unique checkbox or override option takes precedence over component default questions.
2. A new heading is added to the Question Categories tree structure for cases with unique or overridden questions, along with corresponding question sets.
3. To revert overridden components to inherit defaults, go to the Network Component Overrides screen and click "Clear Component Question Overrides."
